Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => app\src\main\AndroidManifest.xml
* res\ => app\src\main\res\
* src\ => app\src\main\java\
* src\._.svn => app\src\main\resources\._.svn
* src\._com => app\src\main\resources\._com
* src\com\._.svn => app\src\main\resources\com\._.svn
* src\com\._google => app\src\main\resources\com\._google
* src\com\google\._.svn => app\src\main\resources\com\google\._.svn
* src\com\google\._android => app\src\main\resources\com\google\._android
* src\com\google\android\._.svn => app\src\main\resources\com\google\android\._.svn
